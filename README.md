##Descrição do Projeto
Este projeto é uma aplicação de linha de comando desenvolvida em TypeScript para gerenciar uma biblioteca pessoal. O sistema utiliza uma lógica de arrays paralelos para armazenar dados como títulos, autores, anos de publicação e avaliações.

O objetivo principal é exercitar conceitos de manipulação de arrays e métodos funcionais de alta ordem, como .filter(), .map(), .reduce() e .forEach().

##Funcionalidades:
- Cadastro e remoção de obras.
- Busca inteligente por títulos ou autores.
- Controle de leitura e sistema de notas (1 a 5).
- Relatórios de produtividade (total de páginas lidas e percentuais).
- Agrupamento automático de livros por década de lançamento.

##Tecnologias Utilizadas
- Node.js
- TypeScript


##Como Executar
Siga os passos abaixo para rodar o projeto localmente:

###1. Clonar o Repositório
```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
cd nome-do-repositorio
```

###2. Instalar Dependências
Certifique-se de que possui o Node.js instalado. No terminal, execute:
```bash
npm install
```

###3. Executar o Projeto
Para iniciar o sistema em modo de desenvolvimento (usando o script configurado no package.json):
```bash
npm run dev
```

##Estrutura de Arquivos
- index.ts: Arquivo principal contendo toda a lógica do sistema e estrutura de dados.
- package.json: Configurações de dependências e scripts de execução.
- tsconfig.json: Configurações do compilador TypeScript.
